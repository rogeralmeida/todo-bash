# todo-bash

This is a simple script to manage todo lists locally.

```shell
./todo.sh --add create a install script to add the script to the path
```

```shell
./todo.sh --list
TODO:
1 salesforce
2 should talk to people
3 Play a song
```
```shell
./todo.sh -d 2
TODO:
1 salesforce
2 Play a song
```
